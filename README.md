## [Render](https://render.com/ "https://render.com/")

-   **Free** - \$0
    -   100 GB bandwidth
    -   \$0.10/GB Past initial 100GB. Ingress bandwidth (network requests made by your code) is always free.

    -   100GB bandwidth

    -   1TB bandwidth
